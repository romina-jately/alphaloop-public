# AlphaLoop public telemetry

Static showcase site for AlphaLoop, a self-grading equity research loop. Served via GitHub Pages from this repo.

- `index.html` is the hand-authored showcase page.
- `alphaloop.json` is the sanitized aggregate snapshot, republished nightly by the private repo's CI. It contains run health, rejection counts, and calibration aggregates only. Holdings, tickers, theses, and model-authored text never leave the private repository.

Recommendations only. AlphaLoop never places or manages any position. Not financial advice.
